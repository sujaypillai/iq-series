## 🎥 Session Summary

### 🎬 Executive Introduction

Asha Sharma introduces Foundry IQ and sets the stage for understanding Microsoft's intelligence layer for knowledge-centric AI systems.

### 💬 Tech Talk

Ayca Bas and Farzad Sunavala walk through the idea of a unified knowledge layer for agents, explaining:
- Core components of Foundry IQ
- How retrieval, reasoning, synthesis, and orchestration come together
- The single model-driven experience approach
