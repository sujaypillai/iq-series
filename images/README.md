This folder contains images and visual assets for The IQ Series.

- `banner.png` - Main banner for the repository
- `speakers.png` - Speaker photos composite
- `visuals/` - Doodle summaries and episode thumbnails

  - `E2-recap.png` - Episode 2 doodle summary
  - `E3-recap.png` - Episode 3 doodle summary
  - `episode1-thumbnail.png` - Episode 1 video thumbnail
  - `episode2-thumbnail.png` - Episode 2 video thumbnail
  - `episode3-thumbnail.png` - Episode 3 video thumbnail
